Launch on Chrome — two options

Option 1 · Local file + bookmark (2 minutes, works today)

  1. Save this file somewhere permanent, e.g. Documents/signal-deck/index.html. Don't leave it in Downloads where you might delete it.
  2. Double-click to open in Chrome, or drag the file onto a Chrome window.
  3. Press Ctrl+D (Windows) / ⌘+D (Mac) to bookmark. Name it Signal Deck.
  4. To pin it to your bookmarks bar: Ctrl+Shift+B shows the bar, then drag the bookmark onto it.
  5. Optional — open on Chrome startup: Chrome Settings → On startup → Open a specific page → paste the file path from your address bar (starts with file:///).

Option 2 · GitHub Pages (10 minutes, gives you a real URL for phone + desktop)

  1. Create a free account at github.com/signup.
  2. Click New repository → name it signal-deck → set Public → tick Add a README → Create repository.
  3. Add file → Upload files → drag this file in, renamed to index.html → Commit changes.
  4. Settings → Pages → Source: Deploy from a branch → Branch: main, folder / (root) → Save.
  5. Wait ~60 seconds. Your URL: https://[username].github.io/signal-deck/
  6. Open that URL in Chrome → Ctrl+D to bookmark on desktop. On your phone: open in Chrome → menu → Add to Home screen — it behaves like an app.

Updating later: when I give you a new version, go to your repo → click index.html → pencil icon → paste new content → Commit. Live in ~1 min. Bookmark never changes.
